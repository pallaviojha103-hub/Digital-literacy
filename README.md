## Project Overview

This Digital Literacy Project is created as part of the CSE0001 course. The purpose of this project is to understand the importance of digital skills, online safety, professional communication, and cyber awareness. <br>

As a Student Digital Ambassador, this project helped me explore different digital platforms, create a professional online presence, and learn how to use technology responsibly and effectively. Each task in this project focuses on a different aspect of digital literacy and its practical application.

### Task 1: Digital Literacy Infographic
In this task, I created an infographic using Canva to explain the concept of digital literacy. The infographic includes topics such as safe internet practices, useful digital tools, and email etiquette. It helped me learn how to present information visually in a clear and attractive way.



###  Task 2: Digital Portfolio
I created my digital portfolio using GitHub, LinkedIn, and Kaggle. These platforms help in building a professional online identity. I also created a GitHub profile README and added my education details on LinkedIn.



###  Task 3: Coding & Collaboration Platforms
I explored HackerRank and completed a beginner-level coding problem. I also created a Google Form quiz on digital literacy and analyzed responses using Google Sheets. This task improved my coding and collaboration skills.

###  Task 4: Email Etiquette & Communication
In this task, I wrote two professional emails — one requesting an assignment extension and another applying for an internship. I also created a social media Do’s and Don’ts checklist to promote responsible online behavior.



###  Task 5: Cybercrime Awareness
I prepared a case study on UPI payment fraud and created a prevention checklist. This task helped me understand common cyber threats and how to stay safe online.
